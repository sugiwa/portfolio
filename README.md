# Ryo Portfolio

## 概要
このサイトでは自分の制作物やスキルを紹介しています。また制作物を作成する際やスキルを学ぶ際に得た知識や問題の解決方法をまとめたサイトも運営しており、そのサイトの一部も紹介しています。

## 開発環境
- 言語・フレームワーク
  - React, Nextjs
- ホスティングサービス
  - Vercel

## API
- Contentful
別サイト([Ryo Blog](https://www.ryoblg.com/))にある最新記事の一部のリンクを取得してその記事に飛べるようにしています。
